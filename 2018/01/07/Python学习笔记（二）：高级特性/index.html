在Python中，为了提高效率，我们需要掌握一些高级特性。

切片

故名思议，即为截取。

L[0:2]指的是索引从0开始，在2之前结束。既包括L[0]、L[1]。

可以从负数到零：L[-2:]其中0可以省略，注意这不是循环，只能负数到零。索引为L[-2]、L[-1]。

迭代

只要是可迭代对象，无论有没有下标，都可以迭代。
